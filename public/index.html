How It Works

Use Strandspace Front To Back

1. Pick Or Describe

Choose a stored topic or describe a new construct in plain language.

The landing page uses your local subjects for navigation and the builder for first-pass drafting.

2. Open The Subject

Jump into a Soundspace-style recall page for the chosen subject.

Each subject view loads its own constructs, prompt chips, and explainable recall trace.

3. Refine In Backend

Send the draft straight into the backend editor when it needs review.

The backend stays the editable memory system for saving, benchmarking, and row-level cleanup.

4. Keep Soundspace

Music Engineering still has its dedicated Soundspace page.

That optimized path stays available while the new subject page pattern covers every other topic.
